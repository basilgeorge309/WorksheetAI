# Scribbl — Session 13 Rubric (Editorial Notebook design system)

Styling-only. Replaces the Cal AI soft-shadow system with a ruled-paper/serif
aesthetic. Behavior/logic/navigation 100% unchanged. Seven checks (one per area).

- [ ] **1. theme.ts + RuledBackground + Home.** `constants/theme.ts` is fully
      replaced (colors/radius/border/spacing/type per spec; Georgia via
      `Platform.select`). `components/RuledBackground.tsx` (react-native-svg)
      draws faint ruled lines + a red margin rule. Home: `<RuledBackground />`
      behind content (content sits right of the margin); "Scribbl" in
      `displaySerif`; usage label green at 3/3 → alertRed when low; dashed-border

- [ ] **5. Settings.** `<RuledBackground />`; section groups = bordered

- [ ] **6. Paywall + DeleteAccount modals.** Sheet bg `colors.paper`, top corners

## Cross-cutting
- No `shadowColor/shadowOpacity/elevation` left anywhere except minimal
  checkmark/confetti depth. All touched files import from the new `theme.ts`.
- Serif-italic only for personality (titles/headlines/upload prompt/button
  labels); system sans for data/chrome (dates, badges, settings rows, inputs).
- `tsc --noEmit` passes; iOS bundle exports clean; logic/nav/state diff = 0;
  `selectable={false}` retained; Confetti timing/physics untouched.

## Out of scope
- No logic/API/navigation/edge-function/query changes; no new screens; no serif
  italic for multi-sentence body copy.
