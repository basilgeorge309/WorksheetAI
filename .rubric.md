# Scribbl — Session 9 Rubric (Google OAuth)

Five pass/fail checks. Wire real Google Sign In via Supabase OAuth + expo-auth-session.

## Checks

- [ ] **1. `signInWithGoogle` does the PKCE code-exchange flow.** It builds the
      redirect with `makeRedirectUri({ scheme: 'worksheetai', path: 'auth/callback' })`,
      calls `supabase.auth.signInWithOAuth({ provider: 'google', options: {
      redirectTo, skipBrowserRedirect: true } })`, opens
      `WebBrowser.openAuthSessionAsync(data.url, redirectTo)`, parses the `code`
      from the success URL, and calls `supabase.auth.exchangeCodeForSession(code)`.
      Returns `{ data, error }`; `cancel`/`dismiss` → `{ error: { message: 'cancelled' } }`;
      never throws. No native module — works in Expo Go.

- [ ] **2. Google button is wired with loading + error handling.**
      `StepFreetier` Google button calls the Google handler; while in flight it
      shows an `ActivityIndicator` in place of the label (button stays white with
      #E5E5E5 border, same size) and all auth buttons are disabled. `'cancelled'`
      is silent (no error shown); any other error shows inline in #DC2626. On
      success it sets AsyncStorage `onboarding_complete = 'true'` and navigates to
      `/(tabs)/`. Email/password and the Apple stub are unchanged.

- [ ] **3. `app.json` has the deep-link scheme + Android intent filter.**
      `scheme` is `worksheetai`; `android.intentFilters` includes a VIEW filter
      (autoVerify true, categories BROWSABLE + DEFAULT) with data
      `scheme: worksheetai`, `host: auth`, `pathPrefix: /callback`.

- [ ] **4. Settings shows the signed-in email above Subscription.** `settings.tsx`
      renders an ACCOUNT label + `user.email` (fontSize 14, #6B6B6B,
      paddingVertical 12) above the existing Subscription section. The existing
      Sign out button still works and the subscription section is intact.

- [ ] **5. Compiles, scoped, documented.** `tsc --noEmit` passes (strict); the
      iOS bundle exports clean; zero inline styles; no new colors; the Apple stub,
      email/password flow, and edge function are untouched. The Supabase + Google
      Cloud dashboard checklist is printed.

## Explicitly out of scope this session
- No Apple Sign In implementation, no Twitter/GitHub OAuth
- No email/password flow changes, no account-linking UI
- No edge function changes
